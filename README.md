# Coding Exercies
Below are links to the Google Colab notebooks for the coding exercises.

**To run:**
- Click the `Open in Colab` button.
- If not already, sign into Google (Top right of screen)
- Click the `Copy to Drive` button to make a Copy of the Colab file (Required if you want to save changes!)
- To run a cell either click the ![Button icon](Data/Images/button_image.png), or use `Ctrl + Enter`
- You can select to `Run all` but we recommend working your way cell by cell so you understand the code.
- To focus on the cell that is currently running
  Click the down arrow of the resources info panel ![RAM/Disk Icon](Data/Images/resources_image.png) and select `Focus the last cell run`.
